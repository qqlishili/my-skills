## 三、决策模式提炼

### 买入决策的前置条件

**体系三要素必须同时有利**（即"三要素共振"）：

1. **竞争格局的比较优势**：必须有明确的产业方向/政策方向
   - 判断标准：场外宏观变化、经济规律、大势所趋
   - 案例：HW产业链（国产替代）、内需消费（央妈提CPI）、商业航天（军民融合）
   - 必须有"师出有名"，合理的说辞与合情的时间窗口

2. **流动性辩证分析**：必须有流动性充裕或增量
   - 判断标准：聚焦人气流动性，是否有溢价效应
   - 拆解来源：央行->证券基金二级市场；ETF基金增量；游资行为
   - 特别警惕：流动性分化、减量博弈、基金赎回压力

3. **情绪位置的变化**：必须在情绪有利位置
   - 判断标准：冰点转折或日内恐慌转折
   - "冰点的冰点发展美，转势"
   - 必须在风险减弱的节点，而非风险增加的节点

**附加条件**：
- 「买入不败」标准：买入时必须考虑明日的情况，是否情绪有利、流动性充裕
- 不做套利和抄底（反复强调不建议做）
- 拒绝博弈分时，只做躺赢阶段
- 「等待，耐心等待，情绪转折向上的时候事半功倍」

### 卖出决策的触发条件

1. **情绪高位转折**：情绪从天而降，流动性分化
2. **竞争格局变化**：方向开始脱离主线，变成虚空炒作（如HW->龙飞凤舞）
3. **风险节点来临**：如514魔咒、宏观风险演绎
4. **亏钱效应出现**：情绪标亏钱效应引导恐慌
5. **一致性风险**：情绪一致性后转折下降（先卖套利标，情绪标减仓一半）
6. **估值透支**：「时间未到，股价先行，注定一路颠簸」（如光弘科技）

### 空仓的决策逻辑

**空仓的定义**（2023-11-11专文阐述）：
「空仓:并不是仓位的空仓。而是指市场交易行为后，个人认定市场不切合自身模式或者体系不利时的空仓。」

**空仓的触发条件**：
1. **存量博弈机构偏移**：如2023上半年AI行情，所有非AI板块都是血包，应空仓
2. **游资行情结束**：炒作完全脱离基本面，进入虚空炒作阶段
3. **减量博弈**：流动性枯竭，闪崩核按钮频繁
4. **增量市场赢太容易**：也可以空仓——「你的实力，不配拥有这种运气」
5. **重要事件窗口前**：如下周有重大事件，可以先空仓等落地

**核心理念**：「规避风险，本质上是规避市场行为的不利。尊重市场，才活的久远。」

### 仓位管理策略

完整框架（2023-08-14《分仓》）：

| 行情状态 | 仓位策略 |
|----------|----------|
| 行情好（三要素共振） | 较大仓情绪标 + 较小仓弹性最大套利标 |
| 行情不好 | 少做/不做，出手都是试错轻仓 |
| 情绪一致性后转折下降 | 情绪标减仓一半 |
| 一致性风险出现 | 先卖出全部套利标 |
| 接近风险节点(如514) | 减仓到盈利持仓 |

### 承认的决策错误及修正

1. **联诚精密天地板（2018）**：最高板卡位失败，两天亏30%
   - 错误：对抗市场，纯资金博弈
   - 修正：追求买入不败，从不过度重仓单一个股

2. **光弘科技割肉（2023）**：看好但市场不利时继续持有
   - 错误：持仓也是买，持仓等不如空仓等
   - 修正：情愿空仓等，也不会持仓等

3. **中韩半导体ETF卖飞（2026）**：没预测到韩国牛市
   - 承认：「我也很菜的」
   - 态度：坦然接受无法预测所有机会

4. **盈利后飘飘然（早期）**：因盈利可以有条件去亏损而放松
   - 修正：越发苛刻，追求买入不败，再也没有大回撤

5. **长电科技发帖时机问题**：「由于发帖相对高位，过山车3次」
   - 导致了大量粉丝挨骂
   - 最后一次建仓在32-33元，坚持长期主义

### 核心方法论特征

1. **宏观先行**：始终从宏观经济学、政治经济学、历史行情演绎三个维度综合判断
2. **三要素框架**：竞争格局 > 流动性 > 情绪位置，三者必须同时有利才行动
3. **风险第一**：先看风险节点，再看机会。"风险在前，面对风险，交易风险的过程。"
4. **等待能力**：大多数时间在等，只等绝佳机会才出手
5. **分仓纪律**：永不全仓一个股，行情好时重仓分仓，行情不好时空仓轻仓
6. **买入不败**：追求最苛刻的买入条件，从源头控制风险
7. **长中短结合**：长线（长电科技/江淮）+ 中线（西部矿业/有色周期）+ 短线（情绪体系交易）

> 「投机是人性扭曲的封建史，投资是国运红利的分配史。」
> 「交易坦坦荡荡，命运明明白白。」

### R5 关键发现总结

**一、"初心不变、载体更替"成为新的认知模板**

R4 中最显著的认知进化是"信念优先于技术"（江丰电子顶背离主动买亏），R5 中出现了新的认知模板："初心没变，承载的个股变了。"这不是在否定初始判断，而是在承认同一方向下需要动态更换标的。从"死守"到"更换载体"，是投资灵活性的重大提升。

**二、宏观择时从"模糊窗口"进化为"精确日期"**

早期以月为单位划定风险窗口（如"4/8-5/14"），R5 进化为精确到交易日的时间标定（"5/27 风险开始、6/12 风险减弱、6/15 风险结束"），且用六重信号交叉验证，精度显著提升。

**三、产业轮动能力从隐到显**

早期三要素框架侧重单方向分析（竞争格局+流动性+情绪），R5 首次系统披露了 A→B 方向的主动产业轮动：识别 A 方向泡沫（材料端有价无市、交易拥挤）→找到 B 方向健康回调（锂电有色有业绩支撑）→执行切换。这是一项此前未显性化的能力。

**四、估值方法向量化演进**

从东材科技"800 亿市值目标→清仓"到西部矿业"4pb 减仓合情合理"，R5 中出现了更多可量化的估值锚点。这与 R4 中出现的"紫金跌破 28 杠杆出清"形成连续趋势——估值判断正在从定性框架向定量模型演进。

**五、"不补仓只做 T"的纪律化表达**

R5 中首次明确声明"基本不补仓"，并给出了完整的逻辑链条（补仓→重仓→杠杆→爆仓）。这与早期偶有"调整配置加仓"的模糊表述形成对比——并非行为改变，而是表述的纪律化。

**六、组合级业绩展示——从个股到系统**
