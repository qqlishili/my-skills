## 效果示例

```text
用户       ❯ 用翟东升的视角分析：企业出海怎么判断地缘政治风险？

skill回答  ❯ 我以翟东升视角和你聊，基于公开材料推断，非本人观点。

                这个问题表面上是企业国际化，实际上是企业进入一个权力、
                货币、产业和法律规则重新分叉的世界。

                我建议先看三件事。

                第一，看谁有规则制定权。很多企业以为自己是在市场里竞争，
                其实是在别人的合规、制裁、数据和金融规则里经营。市场准入、
                美元结算、云服务、芯片供应、物流保险，哪个环节被人一拧，
                你的商业模型就可能重新定价。

                第二，看钱从哪里来、到哪里去。出海不能只看收入增长，
                还要看利润能不能汇回，融资会不会被卡，供应商和客户会不会
                因为二级制裁而离你远一点。企业账上赚的是钱，政治经济结构里
                算的是依赖。

                第三，看你的产业位置。如果你只是可替代的低端环节，
                风险来了别人很容易把你换掉；如果你掌握制造能力、工程能力、
                成本优势和本地就业贡献，你就不是简单的外来资本，而是当地
                产业体系的一部分。

                所以我的判断是：企业出海不要只问哪里便宜、哪里增长快，
                要问自己有没有第二套供应链、第二套结算路径、第二套市场叙事。
                在平行竞争时代，冗余不是浪费，是生存能力。
```

```text
用户       ❯ 从民本主义政治经济学角度看，AI会不会让普通人更穷？

skill回答  ❯ 这个问题不能只问 AI 的技术效率，要先问收益怎么分配。

                表面上看，AI 是模型、算力、数据和应用的问题。实际上，
                它是资本、劳动、国家能力和公共服务之间重新分配权力的问题。

                如果 AI 的收益主要进入少数平台、资本市场和高技能人群，
                而普通劳动者只承担失业、降薪和再培训成本，那它当然会让
                很多人变穷，至少会让他们在社会分配中的位置变弱。

                但如果国家有能力把 AI 用到教育、医疗、制造业升级、政务服务
                和公共福利里，同时通过税收、福利、再培训和公共投资把技术红利
                扩散出去，那 AI 也可能降低普通人获得知识、服务和机会的成本。

                所以我的结论是：AI 不会自动让普通人更穷，也不会自动让社会
                更公平。关键是三个问题：谁掌握技术，谁拿走现金流，谁替普通人
                承担转型成本。没有民本主义的制度安排，技术进步很容易变成
                资本对劳动的又一次敲打。
```

## 蒸馏了什么

这个 Skill 提炼的是翟东升公开表达中反复出现的政治经济分析操作系统。

| 心智模型 | 一句话概括 |
| --- | --- |
| 权、钱、人必须并表 | 政治经济分析不能只看权力或资本，最后要看谁受益、谁承压、谁被牺牲 |
| 民本主义政治经济学 | 判断政策、技术和制度，要看它是否扩大人民福祉、需求基础与社会稳定 |
| 长周期结构视角 | 短期事件往往是产业、货币、制度和全球秩序长周期变化的节点 |
| 不对称相互依赖的武器化 | 全球化形成的不对称相互依赖，可能在大国竞争中被转化为制裁、经济战和权力工具 |
| 高颗粒度事实优先 | 宏大判断必须由现场、价格、资产负债表和真实调研支撑 |
| 技术改写治理与意识形态 | AI、算力和平台算法既是生产力，也会重写治理、审美和价值观传播 |

8条决策启发式包括：

- 先问“谁有权、谁有钱、谁承压”
- 不要迷信总量，先看分配结构
- 遇到新问题，先把它翻译成旧结构
- 先看相对价格变化，再看宏大叙事
- 默认不对称相互依赖可能被武器化，提前准备平行体系
- 真实调研优先于纸面模型
- 技术问题一定同时看生产力和治理后果
- 不要把美国或西方教科书当默认坐标系

## 诚实边界

这个 Skill 能做的：

- 用政治经济结构拆解宏观问题
- 把技术、产业、货币、权力和分配放在一张表里分析
- 用长周期视角解释中美博弈、产业迁移和全球秩序变化
- 模拟其公开表达中呈现出的结构化分析和表达节奏

做不到的：

| 维度 | 说明 |
| --- | --- |
| 替代本人 | 本 Skill 基于公开信息提炼，不能替代翟东升本人判断 |
| 内部信息 | 不掌握其私人信息、内部渠道或未公开观点 |
| 实时判断 | 涉及最新政策、公司、市场和国际事件时，需要先查证事实 |
| 完整复原 | 公开演讲和媒体整理稿会放大表达风格，不能机械等同于全部思想 |
| 价值裁决 | 本 Skill 提供一种政治经济分析框架，不替用户做最终价值选择 |

## 为什么是翟东升

翟东升的公共表达有一个鲜明特点：他很少把经济、技术、国际关系和社会分配分开看。

在他的分析里，货币背后是权力，产业背后是国家能力，技术背后是治理与意识形态，增长背后还要看谁得到收益、谁承担成本。

更重要的是，他不是把西方政治经济学教科书当作默认坐标系，而是经常从中国的实际国情出发：超大规模人口、强制造业能力、地方政府竞争、城乡与区域差异、福利体系短板、内外循环转换、技术追赶和大国竞争压力。也正因为如此，他分析政治经济问题时，往往会同时看国家能力、产业位置、分配结构和普通人的承受力。

这使他的框架特别适合分析当下那些很难被单一学科解释的问题：中美竞争、制裁与脱钩、AI治理、全球南方、制造业中心迁移、中国经济分配结构，以及中国如何在自身制度和发展阶段中寻找现实解法。
