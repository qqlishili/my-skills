# 05 决策与行动线索（G1 Phase 2 汇总）

## 输入隔离声明

本文件仅综合同目录 `README.md` 与 `batch-01.md` 至 `batch-05.md`，未读取任何其他文件、文章原文、Web 或 MCP。证据路径均相对 `references/sources/articles/`；文中动作和结果均是文本记录，未作账户或外部事实核验。

## 1. 买入前置于卖出

- **作者观点**：T+1 下先问次日能否退出。证据：`2023-07-20 151712_冰冰小美_T+1的思考.md`，摘录：“今日的买入，明天能不能顺利卖出去”。
- **作者观点**：流动性有利时更适合卖出。证据：`2023-06-20 155033_冰冰小美_常见的亏钱认知，第十期，会卖的是师傅，会买的是徒弟.md`，摘录：“流动性有利……最好卖”。
- **作者观点**：买入理由应能被复述和检验。证据：`2026-05-21 205540_冰冰小美_还有啥疑问？？？？.md`，摘录：“为什么买，你要有理有据”。
- **汇总推断**：行动起点是可解释的入场条件与退出空间，而不是买后再寻找理由。

## 2. 风险窗口触发仓位降档

- **作者观点**：体系不利时空仓。证据：`2023-11-11 091236_冰冰小美_情绪体系交易篇，第十一期，空仓.md`，摘录：“体系不利时的空仓”。
- **文本事实**：接近风险窗口时减至盈利仓。证据：`2026-05-08 100438_冰冰小美_这个路径明白后，那么根据宏观窗口，即48-514，接近514，突破新高，减仓到盈利持仓。那么立于不败之地。.md`，摘录：“减仓到盈利持仓”。
- **文本事实**：在经验不足时选择控仓等待。证据：`2026-06-08 233850_冰冰小美_聊天完毕。我也很迷茫。经验为零的阶段。从传统风险演绎，我要控仓静待风险释放。从Ai对投资的改造，我要买入Ai指引的方向。.md`，摘录：“控仓静待风险释放”“我没有结论”。
- **汇总推断**：宏观/流动性风险主要调整仓位和节奏，不必自动推翻长期方向。

## 3. 风险预算与心理承受力绑定

- **文本事实**：超过 20% 被作者称为“超级重仓”，减仓常对半。证据：`2026-06-17 120256_冰冰小美_按我的持仓配置，很大程度与ETF持仓配置差不多。超过20%都是超级重仓。同一个方向买入40%也是超级重仓。这一点当初科普.md`，摘录：“超过20%都是超级重仓……减仓每次都是对半减仓”。
- **作者观点**：睡眠恶化表明仓位与心态不匹配。证据：`2026-06-17 112131_冰冰小美_如果被套的怎么办。1，股票波动在所难免。被套第一件事，审问内心，睡眠如何。如果睡眠下降，则是仓位与心态不匹配。你买之前就.md`，摘录：“减仓到睡得着”。
- **作者观点**：风险来临时要有现金余地。证据：`2026-06-11 155931_冰冰小美_一切一切，你得在风险来临时，仓位可控。到市场有打折的时候，风险减弱的节点，你有钱购买。闲钱炒股重要性。很多时候，身不由己.md`，摘录：“仓位可控”“一半中长线，一半短线”。
- **汇总推断**：仓位是认知、波动承受力和后续选择权的共同约束，不只是收益放大器。

## 4. 建仓、持有与退出

- **作者观点**：建仓偏左侧、阶梯式，并先考虑极端风险与三年变化。证据：`2026-06-17 213822_冰冰小美_有人问，我什么时候右侧加仓。我建仓通常是左侧，核心是底线思维，思考极端风险，以及未来三年的变化。因此，建仓通常是阶梯式建.md`，摘录：“建仓通常是左侧……阶梯式建仓”。
- **文本事实**：长期方向仍看好但预期已提前计价时清仓。证据：`2026-06-22 131120_冰冰小美_东材科技 ，800亿市值目标达到。清仓。扩张还没有投产，按投产的盈利，市值先到了。树脂占比pcb成本多少？pcb增长多少.md`，摘录：“按投产的盈利，市值先到了。清仓”。
- **文本事实**：原方向不变而承载标的变化时执行纠偏。证据：`2026-06-22 101011_冰冰小美_亏的时候肯定存在的，核心是纠偏呀。我买那么多个股，有亏有赚的。因为观察我国困境，出海是历史经验。买入的节点就是观察出海利.md`，摘录：“初心没变，承载的个股变了。所以，我纠偏了”。
- **汇总推断**：长期主义并非静态持有；其操作包括分批建仓、底仓、做 T、减仓、清仓和换载体，但触发阈值未统一量化。

## 5. 研究能力不足时主动降级

- **作者观点**：不懂个股时使用 ETF。证据：`2026-05-10 112131_冰冰小美_了解报团基金，然后了解ETF基金。那么重合的权重个股，就是基金重仓股。基金行业过热，或者又开始新发行基金，这一类权重个股.md`，摘录：“抛弃个股选择，选择ETF即可”。
- **作者观点**：ETF 表达方向，不等同于个股。证据：`2026-06-25 084517_冰冰小美_ETF并不是个股，只是方向。你选择你认为对的方向，但是不知道选股，就买ETF，错的概率极小。ETF一篮子股票，肯定有不涨.md`，摘录：“ETF并不是个股，只是方向”。
- **作者观点**：AI 用于初筛后仍需人工判断。证据：`2026-05-18 162734_冰冰小美_我闺蜜最近用AI指导交易，给AI喂数据，然后根据她的专业能力，又再次人工筛选。这成功率高的离谱。但是很多情况，买了也睡不.md`，摘录：“又再次人工筛选”。
- **汇总推断**：能力边界下的降级路径是“个股 -> ETF”“自动初筛 -> 人工复核”“交易 -> 继续观察”。

## 6. 风险排除项

- **作者观点**：不以跌幅本身抄底，尤其要排除造假。证据：`2026-05-21 200236_冰冰小美_估计我才啥都敢说。不过长期主义是允许股价波动的。一定要睁大眼睛，现在的风险不是短线的节奏导致亏损。而是打击造假。千万别踩.md`，摘录：“不要看跌很多的随便抄底”。
- **作者观点**：风险窗口不应满仓。证据：`2026-05-15 142924_冰冰小美_短期风险提过514节点。这个节点还满仓ALL IN那么文章白看。但是结构性行情，也有有利的方向。真正的大行情，需要风险.md`，摘录：“这个节点还满仓ALL IN那么文章白看”。
- **作者观点**：杠杆与补仓求快速回本会形成恶性循环。证据：`2026-06-17 112131_冰冰小美_如果被套的怎么办。1，股票波动在所难免。被套第一件事，审问内心，睡眠如何。如果睡眠下降，则是仓位与心态不匹配。你买之前就.md`，摘录：“这种恶性循环的爆仓比比皆是”。
