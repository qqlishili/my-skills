## 一、一句话

> 周期股在杠杆推动的暴涨后，可通过历史节点回溯找到**杠杆出清的量化参考线**——跌破该线意味着做空动能衰减，基本面不变仍是好股票。

## 二、来源案例：紫金矿业

### 2.1 编年体时间线

| 时间 | 价格 | 事件 |
|------|------|------|
| 2025-10-08 | 28 元（阶段起点）| 杠杆资金推动 |
| 2025-11-24 | 28 元 | 短暂回踩 |
| 2025-12 | 45 元（顶部）| 突破万亿市值，杠杆峰值 |
| 2026-01-03 | — | 白银期货引发恐慌抛售 |
| 2026-03 | 27 元（破位）| **杠杆出清线** |
| 2026-06 | 持续阴跌 | 等待风险减弱 |

### 2.2 杠杆出清线判断

- **关键阈值**：28 元（= 10-08 阶段起点）
- **跌破 28 元的含义**：10-08 涨幅全部抹去 = 杠杆资金全部出清
- **跌破后的判断**：
  > "紫金矿业跌破 28，就意味着杠杆出清了。做空动能下降。那么只需要等靴子落地，基本面依旧优秀，依旧增长，那么，依旧是好股票的。"
  > —— 2026-06-10 134129

## 三、量化锚点的方法

### 3.1 找杠杆出清线的 3 步法

1. **找到上一轮杠杆启动点**（如紫金 10-08 的 28 元）
2. **找到杠杆峰值**（如紫金 12 月的 45 元）
3. **跌破启动点 = 杠杆出清**（28 元是分水岭）

### 3.2 验证

- 跌破后**短期可能继续阴跌**（恐慌情绪未消）
- 但**长期来看是买点**（基本面未变）
- 西部矿业类似：16 以下建仓不亏（=西部矿业 41→29 后，跌破 16 是杠杆出清）

## 四、适用范围与边界

### 4.1 适用

- **周期股**（有色/化工/能源）——杠杆驱动型
- 有明确的"杠杆资金"痕迹（融资余额、龙虎榜机构）
- 历史上有清晰的"启动点→峰值→下跌"三段

### 4.2 不适用

- **成长股**（如 AI 产业链）——无杠杆周期
- **消费/金融股**——价值投资逻辑，不是杠杆逻辑
- **黑天鹅事件**（如财务造假）——不是杠杆出清，是清零

## 五、与启发式 16「流动性预期 ≠ 企业经营」配套

- 启发式 15：**识别杠杆出清的量化线**（如紫金 28 元）
- 启发式 16：**理解下跌原因**（杀流动性 vs 杀逻辑）
- 两者关系：跌破杠杆出清线 + 企业经营无变化 = 买点（启发式 15+16 双重验证）

## 六、应用流程

```
1. 标的持续下跌
2. 问：这是周期股吗？是否有杠杆资金痕迹？
   ├─ 否 → 用其他启发式
   └─ 是 → 进入第 3 步
3. 找到上一轮杠杆启动点（K 线/融资数据）
4. 当前价是否跌破启动点？
   ├─ 否 → 继续观察，可能未出清
   └─ 是 → 启动启发式 15
5. 验证：企业经营是否变化？
   ├─ 是变化 → 启动启发式 17 止损
   └─ 无变化 → 启动启发式 16 区分杀流动性
6. 决策：分批建仓 + 设止损
```
