# references/heuristics/legacy/h14-industry-belief.md — 历史模板

> 启发式 14「产业信念优先于技术信号」深度展开
> 创建时间：v1.3 | 来源：research/05 第七章 H-17

## 一、一句话

> 当企业发生根本性竞争力变化（如进入全球顶级供应链、获得国际认证），技术面的顶背离/破位不应阻止买入——**信念优先于技术**。

## 三、边界条件（关键！）

启发式 14 **不是**死扛规则。**有 3 个边界**：

### 3.1 适用边界

- **中长线产业标的**（非短线情绪标的）
- 企业有**根本性竞争力变化**（不是简单的"业绩好"）
- 变化可被**外部验证**（如供应链认证、专利、客户公告）

### 3.2 不适用边界

- **短线情绪标的**（妖股/题材股）——技术面破位仍应止损
- **无产业逻辑的"价值股"**——技术面破位可能真的是基本面恶化
- **黑天鹅事件**（如财务造假曝光）——信念立即清零

### 3.3 自我和解过程

- 基线 153 篇明确"排斥纯技术"——但 5-19~6-11 出现"主动买亏"
- 解决方案：**分层**——对纯情绪标的，技术/情绪止损依旧有效；对产业确定性标的，技术面可以让步

## 四、与启发式 17「主动买亏的边界」的关系

- 启发式 14 是"信念优先于技术"——**主动买亏的勇气**
- 启发式 17 是"什么时候该认输"——**基本面证伪的止损纪律**
- 两者**配套使用**：14 让你敢买，17 让你知道何时止损

## 五、应用流程

```
1. 标的出现技术破位（顶背离/跌穿均线）
2. 问：这是短线情绪标的还是中长线产业标的？
   ├─ 短线 → 用启发式 3/7（亏钱效应/减少出手）
   └─ 中长线 → 进入第 3 步
3. 问：企业近期是否有根本性竞争力变化？
   ├─ 无 → 技术破位可能是真的，按纪律止损
   └─ 有（外部可验证）→ 用启发式 14，加仓或持有
4. 持续监控：基本面逻辑是否被证伪？
   ├─ 否 → 持有（允许技术面不利）
   └─ 是 → 启动启发式 17 止损
```
