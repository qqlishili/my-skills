# 📊 Stock Deep Analyst

**A股个股业绩深度分析与投资策略生成** — WorkBuddy AI Agent Skill

一个标准化的个股深度分析框架，帮助你在几分钟内生成专业级投资分析报告。

---

## ✨ 功能特性

- 🔍 **全面数据采集** — 自动获取行情、公告原文、财务报表、技术指标、筹码、资金流向、机构预期等 11 类数据
- 📈 **10 大分析板块** — 标准化覆盖业绩拆解、估值评估、技术面、资金面、风险矩阵等
- 🎯 **多维度估值** — PE/PEG/PS/PB/DCF 等 8 种估值方法交叉验证
- 🧑‍💼 **分人群策略** — 针对重仓/轻仓/未持仓-积极型/未持仓-稳健型 4 类投资者分别给出操作建议
- ⚠️ **风险量化** — 概率 × 影响度矩阵评估 6 大风险因子
- 📋 **质量自检** — 内置 10 项 Checklist 确保报告完整性

## 🔧 安装

### 前置要求

- [WorkBuddy](https://www.codebuddy.cn/) IDE 或 WorkBuddy ClawBot
- 已安装 `westock-data` Skill（用于获取 A股/港股/美股行情数据）

### 安装方式

将本仓库克隆到 WorkBuddy Skills 目录：

```bash
git clone https://github.com/koopa1025/stock-deep-analyst.git ~/.workbuddy/skills/stock-deep-analyst
```

安装完成后，WorkBuddy 会自动识别该 Skill。

## 🚀 使用方式

安装后，对 AI 助手说以下类似的话即可自动触发：

| 场景 | 示例指令 |
|------|---------|
| 业绩预告分析 | "沪电股份发了 Q1 业绩预告，帮我深度分析一下" |
| 年报 / 快报分析 | "比亚迪年报出了，分析一下" |

| 持仓决策 | "我持有贵州茅台，现在该怎么操作？" |

## 📊 分析框架（10 大板块）

| # | 板块 | 核心内容 |
|---|------|---------|
| 1 | 业绩核心数据 | 营收/净利/EPS/ROE 等指标对比表 |
| 2 | 季度拆解 | 单季度趋势、环比增速、季节性判断 |
| 3 | 超预期分析 | 实际 vs 机构一致预期，偏离度量化 |
| 4 | 估值分析 | PE/PEG/PS/PB + 合理价值计算（≥3 种方法）|
| 5 | 技术面分析 | 均线/MACD/KDJ/RSI/BOLL/CCI 综合判断 |
| 6 | 资金面分析 | 主力净流入、超大单/大单/中单/小单拆解 |
| 7 | 筹码分析 | 获利比例、平均成本、集中度 |
| 8 | 机构观点 | 最新研报汇总、一致目标价、多空分歧 |
| 9 | 风险矩阵 | 6 大风险因子 × 概率 × 影响度评分 |
| 10 | 投资策略 | 4 类投资者操作方案 + 止损 + 敏感性分析 |

## 📖 报告示例

以下是使用本 Skill 生成的报告片段：

```
【一句话结论】
沪电股份2026Q1业绩预告归母净利+55%~65%，显著超预期。
中长期看 ¥120-140，但当前 ¥97.54 短期超买（RSI=99.69）。
最优策略是等回调至 ¥85-93 分批建仓。
```

## 🎨 规范

- **涨跌颜色**：遵循 A 股惯例 — 🔴 红色 = 涨，🟢 绿色 = 跌
- **货币单位**：人民币 ¥，标注"元"或"亿元"
- **数据精度**：金额 2 位小数，百分比 2 位小数
