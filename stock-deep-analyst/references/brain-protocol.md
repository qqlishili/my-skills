# 爆金总脑认知协议（核心机制摘要）

> 本文件是爆金总脑协议的核心机制摘要，供独立使用本 Skill 的 Agent 加载。
> 完整协议包含 21+ 条机制，此处仅摘录与个股深度分析直接相关的核心机制。

---

## 一、定位

爆金总脑是投资认知的结构化、可持续进化的操作系统内核。它不是全自动量化系统（以主观认知为核心），不是研究搬运工（有自己的判断框架），必须持续迭代、校准、进化。

---

## 二、V1.1 六条防失真机制

### 机制 1：负债端视角
不只看资产端（公司好不好），还要看持有人结构和定价锚。任何持仓讨论，除了看基本面，还要问"谁在持有、谁在定价、资金结构是否脆弱"。

### 机制 2：全球产业锚点机制
淡化中国特色叙事，回到全球产业普适规律找锚。讨论 A 股公司时，先找全球对标和全球竞争格局，再判断中国特色溢价/折价。

### 机制 3：网格验证机制
科技产业不能只做链式推演，要多节点交叉验证。关键判断至少找 2-3 个独立验证点。

### 机制 4：景气/主题分流机制
先分清"真景气"还是"主题景气"：
- 真景气 = 有订单、有利润、有持续性 → 可以做趋势持仓
- 主题景气 = 有热度、有想象力、但缺硬验证 → 只适合做弹性博弈

### 机制 5：过界检测机制
任何新变化，先判断它属于"边际修正 / 结构变化 / 范式切换"三个级别中的哪一个。

### 机制 6：风险分层处置机制
- 短期可逆风险 → 观察为主
- 短期不可逆风险 → 立刻降风险
- 中期结构风险 → 有序降权重
- 长期范式风险 → 系统性重新评估

## 三、财报阅读认知框架（机制 7-12）

**总原则：财报是用来证伪的，不是用来证实的。** 看财报的第一目标不是找亮点，而是排雷。

### 机制 7：扣非利润优先机制
默认看扣非净利润，而不是归母净利润。归母和扣非差异越大，利润质量越需警惕。

### 机制 8：季度节奏追踪机制
必须拉出最近 8 个季度的单季利润，画出节奏线。同比消除季节性，环比看短期边际变化。

关键判断组合：
- 同比加速 + 环比加速 = 最强信号（景气确认）
- 同比加速 + 环比走平 = 正常（稳步改善）
- 同比走平 + 环比下降 = 预警（增速可能见顶）
- 同比下降 + 环比下降 = 警报（需重新评估）

### 机制 9：利润-现金流交叉验证机制
- 经营现金流净额 / 净利润 ≥ 1 → 利润质量好
- < 0.7 且持续多季 → 利润可能"虚胖"
- 经营现金流持续为负但利润在增长 → 高度警惕

### 机制 10：应收账款与存货周转效率机制
- 应收增速持续高于营收增速 → 可能在放松赊销条件冲收入
- 应收账款占营收比例超 30% 要警惕，超 50% 要降级
- **DSO 增速差判读**：应收余额同比增了 X%，但周转天数同比缩短了 → 收款效率在提升 → ✅ 健康
- **DIO 增速差判读**：存货余额增了 50%，但周转天数只增了 20% → 产品在加速出货 → ✅ 健康

### 机制 11：分业务穿透机制
多业务公司必须穿透到各业务板块单独评估：收入、增速、毛利率、收入占比变化。判断"核心引擎"是谁。

### 机制 12：财报三表联动检查机制
- 利润表看趋势
- 现金流表验真伪
- 资产负债表看质量

核心交叉检查：
1. 利润增长 + 经营现金流同步增长 → ✅ 健康
2. 利润增长 + 经营现金流下降 → ⚠️ 利润质量存疑
3. 利润增长 + 应收大幅增加 → ⚠️ 回款风险
4. 利润增长 + 存货大幅增加 → ⚠️ 可能备货或滞销

### 机制 12-B：现金转换周期（CCC）追踪机制
- CCC = 应收周转天数 + 存货周转天数 − 应付周转天数
- CCC 缩短 → 营运效率在提升 ✅
- CCC 延长 → 资金在链条上停留更久 ⚠️
- 分解排查：哪个环节出了问题（DSO/DIO/DPO）

## 四、估值范式核心机制（机制 21-22）

### 机制 21：交汇性估值
**适用条件**：多条技术路线在该节点交汇，纵向（产品代际迭代）+ 横向（路线扩展）都绕不开。

**硬度分级**：
- 硬交汇 = 物理层面绕不开（如光芯片、TFLN 调制器）
- 中交汇 = 工艺/量产领先但非独占
- 弱交汇 = 暂时方案优选

**子判断：期货型 vs 现货型**
- 现货型 = 利润已释放、估值已锚定
- 期货型 = 远期利润为主、当前估值看起来很高

### 机制 22：稀缺性估值
**适用条件**：下游稀缺资产。看上游自给率、份额集中度、技术壁垒。

**窗口期判断**：
- 剩 >2 年 → 稀缺性溢价可给 15%-25%
- 剩 <1 年 → 溢价收敛

### 机制 21/22 合成估值法
当一个标的同时具备交汇性和稀缺性时：
1. 先用传统 PE/PEG 算出基础估值
2. 叠加交汇性溢价（硬交汇 +20%-40%）
3. 叠加稀缺性溢价（+10%-30%）
4. 合成估值 = 基础 × (1 + 交汇溢价 + 稀缺溢价)
5. 上限建议：溢价合计不超过 +100%

### 路径博弈估值
**适用条件**：技术/政策路径不确定。不能用简单 PE，按期权仓管理：
- 估值 = 兑现概率 × 兑现后弹性 × 目标市值
- 按期权仓管理：小仓位、高赔率、允许归零

### 传统估值
**适用条件**：业绩稳定、逻辑清晰。用 PE/PEG/DCF。
